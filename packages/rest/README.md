# Cadl Rest Library

This package provides [Cadl](htps://github.com/microsoft/Cadl) decorators to describe REST API.

## Install

In your cadl project root

```bash
npm install @cadl-lang/rest
```

## Usage

See [Rest section in the tutorial](../../docs/tutorial.md#rest-apis)
